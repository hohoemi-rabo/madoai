# 006: AIチャットページUI（Geminiライクデザイン）

## 概要

トップページ（`/`）にGeminiライクなモダン・没入感のある対話型AIチャットUIを実装する。ダークテーマベースで、初期状態のウェルカム画面と会話中の表示を含む。

## 対象（REQUIREMENTS.md §11, §12, §17 Phase A-7,8）

## デザイン方針

- Geminiのトップ画面を参考にした没入感のあるダークテーマUI
- 主役: 初期状態→入力ボックス、会話中→AI回答テキスト
- 不要な要素を削除（＋ボタン、マイク、自治体セレクター）
- Lucide Icons使用（絵文字不使用）
