# 006: AIチャットページUI（Geminiライクデザイン）

## 概要

トップページ（`/`）にGeminiライクなモダン・没入感のある対話型AIチャットUIを実装する。ダークテーマベースで、初期状態のウェルカム画面と会話中の表示を含む。

## 対象（REQUIREMENTS.md §11, §12, §17 Phase A-7,8）

## デザイン方針

Geminiのトップ画面を参考にした、モダンで没入感のあるダークテーマUI。

## タスク

### 1. テーマ・カラーパレット（ダークモード化）
- [ ] 全体をダークテーマに変更
- [ ] 背景色: `#131314`（非常に暗いグレー）
- [ ] テキストカラー: 白〜明るいグレー基調
- [ ] globals.css / Tailwind設定のカラーパレット更新

### 2. ヘッダー
- [ ] 背景を透過 or 黒に変更（ダークテーマ対応）
- [ ] シンプルで目立たないデザイン（Gemini風）
- [ ] サービス名 + ナビゲーション（最小限）

### 3. 初期状態 — メインエリア
- [ ] 画面中央に大きな挨拶メッセージ: 「✨ 何から始めますか？」（左揃え、中央付近配置）
- [ ] 免責バナー（「本サービスは飯田市の公式サービスではありません」常時表示、ダークテーマ対応）
- [ ] 自治体セレクター（MVP: 飯田市固定）

### 4. 入力エリア（チャットボックス）
- [ ] 画面中央（挨拶メッセージの直下）に配置
- [ ] 大きくて角丸の長方形デザイン
- [ ] 背景色: `#1e1f20`（明るめのダークグレー）
- [ ] 左側: 「＋」ボタン / ツールアイコン（プレースホルダーUI）
- [ ] 右側: 「マイク」アイコン（プレースホルダーUI）
- [ ] 送信ボタン

### 5. サジェストチップ（提案ボタン）
- [ ] 入力ボックスの「下」に配置
- [ ] ダークテーマ対応（暗い背景 + 白テキスト）
- [ ] 「アイコン + テキスト」構成のモダンなボタンデザイン
- [ ] 角丸で視覚的に統一
- [ ] 代表的な4〜5個に絞り、1行で表示（横スクロール可）
- [ ] タグクリック → 質問文挿入 → 自動送信

### 6. 会話中の表示
- [ ] 会話エリア（ユーザー質問とAI回答の交互表示、スクロール可能）
- [ ] ストリーミング表示（1文字ずつ流れるように表示）
- [ ] ソースカード表示（参照元ページへのリンク: タイトル + URL + 自治体名、ダークテーマ対応）
- [ ] 問い合わせ情報表示（担当部署・電話番号）
- [ ] 免責注記（「※ 詳細は公式ページまたは担当窓口でご確認ください」）
- [ ] 免責バナーのコンパクト化（折りたたみ可）
- [ ] 会話開始後、入力欄は画面下部に固定移動

### 7. 会話機能
- [ ] sessionStorageでの会話履歴保存
- [ ] 匿名session_idの生成・localStorage保存
- [ ] 直近10往復のコンテキスト送信
- [ ] SSEストリーミングのクライアント側処理

### 8. レイアウト
- [ ] Geminiライクレイアウト（幅広の回答表示、吹き出しなし）
- [ ] ルーティング設定（`/` → AIチャット）
